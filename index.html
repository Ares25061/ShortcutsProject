Горячие клавиши Blender

Этот сайт представляет подборку полезных стандартных горячих клавиш в Blender.

Изменить вид

Средняя кнопка мыши

  • Перетаскивание Средней кнопкой мыши вращает вид. Вращение происходит вокруг невидимого центра интереса.

  • Зажмите Shift и перетаскивайте, чтобы переместить центр интереса.

  • Зажмите Ctrl и перетаскивайте или крутите колесико мыши, чтобы приближать/отдалять вид.

Вид по осям

Num 1 – 9

  • Num 1, Num 3 и Num 7 для вида спереди, сбоку и сверху.

  • Num 9 для инвертирования текущего направления вида.

  • Num 5 для переключения в ортографический режим.

  • Num 2, Num 4, Num 6, Num 8 для поворота вида с шагом в 15°.

Вид из камеры

Num 0

  • Num 0 для выравнивания вида по активной камере.

  • Ctrl + Num 0 для выравнивания вида по выбранной камере и назначения её активной.

  • Ctrl + Alt + Num 0 для выравнивания активной камеры по текущему виду.

Сфокусироваться на выделенном

Num .

  • Num . установить центр интереса на текущее выделение.

  • Num ∕ для переключения локального вида, который также скрывает все, кроме выбранного объекта.

Сменить режим

Ctrl + Tab

  • Ctrl + Tab для отображения всех доступных режимов.

  • Tab для прямого переключения в режим правки.

  • Если курсор наведен на таймлайн, Ctrl + Tab переключает в режим графов (Graph Editor).

Переключить объект

Alt + Q

  • Alt + Q для переключения на объект под курсором без смены режима.

  • Работает во всех режимах, включая режим позы для объектов-арматур.

Переместить

G

  • Нажмите X, Y или Z для фиксации оси. Нажмите дважды для переключения с глобальных на локальные оси.

  • Shift + X, Y или Z для исключения оси.

  • В режиме правки нажмите G дважды для скольжения вдоль существующих ребер.

Повернуть

R

  • Нажмите X, Y или Z для фиксации оси. Нажмите дважды для переключения с глобальных на локальные оси.

  • Shift + X, Y или Z для исключения оси.

  • Нажмите R дважды для переключения в режим карданова подвеса (Gimbal).

Масштабировать

S

  • Нажмите X, Y или Z для фиксации оси. Нажмите дважды для переключения с глобальных на локальные оси.

  • Shift + X, Y или Z для исключения оси.

Добавить

Shift + A

  • Shift + A добавляет новые данные.

  • В зависимости от контекста это может быть новый объект, новая геометрия внутри активного объекта или новый узел.

Удалить

X / Del

  • Нажмите X или Del в объектном режиме для удаления выбранных объектов.

  • Нажмите X или Del в режиме правки для удаления выбранных вершин, ребер или граней.

Скрыть

H

  • H чтобы скрыть выделенное.

  • Shift + H чтобы скрыть всё, кроме выделенного.

  • Alt + H чтобы показать всё скрытое.

  • Геометрия, скрытая в режиме правки, будет видна в объектном режиме.

Дублировать

Shift + D

  • Shift + D для дублирования выделения.

  • Alt + D в объектном режиме для создания связанных дубликатов, которые ссылаются на исходные данные вместо их копирования.

Изменить затенение

Z

  • Shift + Z для переключения Каркаса и Рентгена, чтобы видеть и выбирать сквозь сетку.

  • Alt + Z для переключения только Рентгена.

Режимы выделения

1 , 2 , 3

  • 1 для выделения вершин.

  • 2 для выделения ребер.

  • 3 для выделения граней.

  • Переключение режимов может изменить выделение!

Привязка (меню)

Shift + S

  • Перемещает один элемент точно к другому. Особенно полезно для позиционирования 3D курсора.

  • Shift + Правая кнопка мыши для ручной установки 3D курсора.

Привязка (временная)

Ctrl

  • Удерживайте Ctrl для включения привязки.

  • Работает с Перемещением, Вращением, Масштабированием и ползунками.

  • Можно привязываться к инкрементам (по умолчанию) или к геометрии (вершины, ребра, грани).

Привязка (базовая точка)

B

  • После начала Перемещения, Вращения или Масштабирования, нажмите B для включения привязки по базовой точке.

  • Первый клик устанавливает базовую точку, второй клик — цель привязки.

Очистить трансформацию

Alt + G , R , S

  • Только в объектном режиме.

  • Alt + G для сброса положения.

  • Alt + R для сброса поворота.

  • Alt + S для сброса масштаба.

Применить

Ctrl + A

  • Только в объектном режиме.

  • Ctrl + A для применения изменений трансформации.

  • Применяет модификатор, если навести на него курсор в списке.

Повторить последнее

Shift + R

  • Shift + R для повторения последнего действия.

  • Повторяет Перемещение, Вращение и Масштабирование относительно результата предыдущего действия.

Выделить всё

A

  • A для выделения всех объектов, вершин, ключей кадров и т.д. в зависимости от контекста.

  • Alt + A для снятия выделения.

Добавить к выделению

Shift + Левая кнопка мыши

  • Shift + Левая кнопка мыши по невыделенному добавляет в выделение.

  • Shift + Левая кнопка мыши по выделенному удаляет из выделения.

Выделить кольцо/петлю

Alt + Левая кнопка мыши

  • Alt + Левая кнопка мыши, когда курсор находится рядом с пересекающим ребром.

  • Alt+Shift+Левая кнопка мыши для добавления или удаления петель из выделения.

  • Работает только на множественных гранях, если они четырехугольные ("квадраты").

Выделить путь / область

Ctrl + Левая кнопка мыши

  • Ctrl+Левая кнопка мыши для добавления кратчайшего пути вершин или граней к выделению.

  • Ctrl+Shift+Левая кнопка мыши для добавления или удаления области.

Выделить связанное

L

  • L для выделения всех связанных вершин под курсором.

  • Shift + L для удаления всех связанных вершин из выделения.

  • Ctrl + L для выделения того, что связано с текущим выделением.

Выделить больше / меньше

Ctrl + Num +

  • Ctrl + Num + для выделения большего (расширить выделение).

  • Ctrl + Num - для выделения меньшего (сузить выделение).

  • Ctrl + Shift + Num + для выделения элементов на том же расстоянии, что и последние два.

Выделить похожее

Shift + G

  • Shift + G для выделения всего со схожими признаками с текущим выделением.

  • Также работает с выделенными объектами или узлами (Select Grouped).

Экструдировать

E

  • E для выдавливания связанной копии выделенной геометрии и ожидания ввода для перемещения.

  • ESC или Правая кнопка мыши для завершения выдавливания без перемещения. Создает "дубликаты" (совпадающие вершины)!

  • Alt + E для дополнительных опций.

Вставка граней (Inset)

I

  • I для вставки граней внутрь выбранных.

  • Нажмите I еще раз для индивидуальной вставки каждой грани.

  • Нажмите O для переключения в режим выступа (Outset).

  • Нажмите B для переключения учета граничных граней.

Добавить грань

F

  • F для добавления новой грани в сетку. Создает новое ребро, если выделены две отдельные вершины.

  • Alt + F для заполнения N-угольников треугольниками.

  • Ctrl + F для вызова меню всех опций работы с гранями, включая "Сеточное заполнение" (Grid Fill).

Объединить (Merge)

M

  • Объединить все выбранные вершины в указанном месте.

  • Объединить по расстоянию (Merge By Distance) для удаления вершин, которые находятся ближе, чем пороговое значение (настраивается в меню).

Разрыв (Rip)

V

  • V для разрыва вершин или ребер.

  • Alt + V для разрыва с заполнением (Rip Fill).

  • Направление разрыва определяется положением курсора.

Разделить (Split)

Y

  • Y для разделения выделения (отсоединения вершин/ребер).

  • Alt + M для разделения всех выбранных граней по их ребрам.

  • Выделение отсоединяется, но остается частью того же блока данных и объекта.

Отделить (Separate)

P

  • P для отделения выделения в новый объект.

  • Новый объект будет иметь те же модификаторы, что и объект, от которого он был отделен.

Контурный разрез (Loop Cut)

Ctrl + R

  • Ctrl + R для разрезания кольца граней на сегменты.

  • Прокрутите колесико мыши для изменения количества сегментов.

  • Работает только на множественных гранях, если они четырехугольные ("квадраты").

Соединить путь

J

  • J для соединения вершин в порядке их выделения.

  • Разрезает грани и создает дополнительные вершины, когда путь пересекает существующие ребра.

Снятие фаски (Bevel)

Ctrl + B

  • Ctrl + B для снятия фаски с ребер.

  • Прокрутите колесико мыши для регулировки количества сегментов.

  • V для переключения режима фаски вершин.

  • C для переключения ограничения (clamping).

  • P для регулировки профиля фаски.

Растворить (Dissolve)

Ctrl + X

  • Ctrl + X для удаления вершин, соединяя при этом окружающие ребра. Результат зависит от режима выделения.

  • X > Limited Dissolve растворяет все вершины/ребра ниже порогового угла.

Сжать/Расширить (Shrink/Fatten)

Alt + S

  • В режиме правки, Alt + S для перемещения вершин вдоль их нормалей.

Пропорциональное редактирование

O

  • O для включения/выключения пропорционального редактирования.

  • Прокрутите колесико мыши для изменения радиуса влияния.

  • Shift + O для изменения профиля спада (falloff).

Поиск

F3

  • F3 для открытия поиска операций (Menu Search).

  • Стрелка вверх и Стрелка вниз для навигации, Enter для выбора.

Переименовать

F2

  • F2 для переименования активного объекта.

  • Ctrl + F2 для пакетного переименования нескольких объектов сразу.

Экструдировать к курсору

Ctrl + Правая кнопка мыши

  • Ctrl + Правая кнопка мыши для выдавливания в позицию курсора с подстройкой поворота предыдущего сегмента (удобно для моделирования кривых).

  • Выдавливания будут размещены на глубине 3D курсора.

Точный ввод

Shift

  • Удерживайте Shift, чтобы уменьшить чувствительность мыши для более точного контроля.

  • Работает с Перемещением, Вращением, Масштабированием и ползунками.

Мульти-редактирование

Alt

  • Если выделено несколько объектов, изменение параметров обычно влияет только на активный объект.

  • Удерживайте Alt, чтобы изменить параметры всех выделенных объектов одновременно.

Превратить в сферу

Shift + Alt + S

  • Shift + Alt + S перемещает выбранные вершины, приближая их к сфере с центром в 3D курсоре.

  • Превращает одиночную петлю вершин в окружность.

Модификатор Subdivision Surface

Ctrl + 1

  • Ctrl + 1, 2, 3 для добавления модификатора подразделения поверхности (Subdivision Surface) с соответствующим уровнем.

  • Ctrl + 0 отключает модификатор (уровень 0).
